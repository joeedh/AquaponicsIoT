Common abbreviations:

* IF      :  interface (i.e. API interface)
* DB      :  database (need not be relational)
* serv    :  server
* th      :  theta (e.g. angles)
* f       :  factor (usually floating-point)
* fac     :  same as f
* t       :  same as f, but used in polynomials
* k       :  coefficient
* p       :  point
* co      :  coordinate
* no      :  normal
